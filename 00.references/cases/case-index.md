# 判例素材一覧

## 運用ルール

- 判例解説記事を追加したら、まず `originals/` に原資料を置く。
- Markdown化した本文は `markdown/` に置く。
- 個別分析は `analysis/` に `case_id` ベースのファイル名で保存する。
- 1章1判例の候補として使う場合は、優先度と使用候補巻を必ず記入する。
- まだ章構成案は作らず、この一覧では素材管理に限定する。

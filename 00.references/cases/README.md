# 判例素材PDFの運用ルール

## フォルダの役割

- `inbox/` は未選別PDFの置き場
- `originals/` は採用または処理対象として確定したPDFの置き場
- `markdown/` は抽出・整形後のMarkdown
- `analysis/` は個別分析
- `templates/` は分析・章設計テンプレート
- `99.archive/` は不採用・差し替え済み・別巻候補の退避先

## Codexで新規case化するときの扱い

- Codexが新規case化するときは、原則として `originals/` のPDFを対象にする
- `inbox/` のPDFは、採用候補探索時のみ対象にする
